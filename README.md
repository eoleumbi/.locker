## About

[eoleumbi]'s dotfiles for [Vim], [Git] and so on.

## Quick Start

1. Introduction:

   requires [Git]

2. Set up:

   ``` bash
   $ git clone https://github.com/eoleumbi/.locker.git ~/.locker
   $ echo source ~/.locker/.vimrc > ~/.vimrc
   $ cd ~
   $ ln -s ~/.locker/.bash_aliases
   $ ln -s ~/.locker/.zshrc
   $ ln -s ~/.locker/.gitconfig
   $ ln -s ~/.locker/.screenrc
   ```
